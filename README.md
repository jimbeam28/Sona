# Sona

基于 Flutter 的 Android NAS 音频播放器，通过 WebDAV 协议连接 NAS（飞牛OS / 群晖 / 威联通等），支持音乐和有声书流式播放。

## 功能简介

- **WebDAV 连接** — 添加/编辑/删除 NAS 连接，支持多连接切换，启动自动验证
- **文件浏览** — 目录树导航、面包屑路径、多排序方式（名称/时间）、音频文件过滤
- **流式播放** — 支持 mp3 / flac / aac / m4a / m4b / ogg / opus / wav 格式
- **播放队列** — 顺序播放、列表循环、单曲循环、随机播放四种模式
- **后台播放** — 锁屏和通知栏媒体控件，App 进后台音频不中断
- **迷你播放器** — 浏览页底部常驻播放栏，快速控制播放
- **播放速度** — 0.5x ~ 2.0x 六档调速，可记住速度设置
- **定时停止** — 5/10/15 分钟固定时长、播完当前、自定义时长
- **进度记忆** — 自动保存播放位置，启动恢复，跳过开头(<5s)和结尾(>duration-10s)
- **外观设置** — 跟随系统 / 亮色 / 暗色主题切换

## 使用说明

### 1. 添加连接

首次打开进入引导页 → 点击「添加连接」→ 填写 WebDAV 地址、用户名、密码 → 点击「测试连接」验证 → 验证通过后「保存」。

> **飞牛OS**：文件管理 → 设置 → WebDAV 开启服务
> **群晖**：套件中心 → WebDAV Server 启用
> **威联通**：控制台 → 应用服务 → WebDAV 启用

### 2. 浏览与播放

添加连接后进入文件浏览页，点击文件夹进入子目录，点击音频文件开始播放。播放页面提供完整的播放控制：进度条拖拽、快进/快退、切歌、调速、定时、队列管理。

### 3. 设置

浏览页右上角进入设置，可调整默认播放速度、快进步长、主题模式、管理连接。

```bash
flutter test                 # 运行测试

```

环境要求：Flutter SDK >= 3.3.0、Java 17、Android SDK。
