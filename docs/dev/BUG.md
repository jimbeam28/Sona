## 已修复、待验证的BUG
1. 当前正在播放音乐，在播放单中点击该音乐，无论是点击重头播放，还是等待5秒后自动播放，都会黑屏。屏幕侧滑后直接APP直接退出
2. 最后一曲播放完毕后，无论是正常播放完毕，还是快进至结束的，音乐停止的同时“暂停”按钮未切换至“播放”按钮。
3. 删除播放列表中的歌曲时，点击后并没有移除，反而乱切歌。
